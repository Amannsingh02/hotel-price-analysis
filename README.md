# Hotel Price Analysis (2012–2016)

Analysis of hotel pricing data to identify seasonal patterns, 
outliers, and forecast prices for February 2022.

## Project Structure
- `hotel_price_analysis.ipynb` — Main analysis notebook
- `data/` — Raw CSV data
- `requirements.txt` — Python dependencies

## How to Run
1. Clone this repo
2. Install dependencies: `pip install -r requirements.txt`
3. Launch Jupyter: `jupyter notebook`
4. Open `hotel_price_analysis.ipynb`

## Tools Used
- Python, Pandas, Matplotlib, Seaborn, Scipy, Prophet
